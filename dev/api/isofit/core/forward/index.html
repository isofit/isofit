    Forward

    source module isofit.core.forward

    Classes

    • ForwardModel — ForwardModel contains all the information about how to calculate radiance measurements at a specific spectral calibration, given a state vector. It also manages the distributions of unretrieved, unknown parameters of the state vector (i.e. the S_b and K_b matrices of Rodgers et al.

    source class ForwardModel(full_config: Config, cache_RT: RadiativeTransfer = None)

    ForwardModel contains all the information about how to calculate radiance measurements at a specific spectral calibration, given a state vector. It also manages the distributions of unretrieved, unknown parameters of the state vector (i.e. the S_b and K_b matrices of Rodgers et al.

    State vector elements always go in the following order

    (1) Surface parameters (2) Radiative Transfer (RT) parameters (3) Instrument parameters

    Attributes

    • nbvec — Set up state vector indices - MUST MATCH ORDER FROM ABOVE ASSIGNMENT

    Methods

    • out_of_bounds — Check if state vector is within bounds.

    • xa — Calculate the prior mean of the state vector (the concatenation of state vectors for the surface, Radiative Transfer model, and instrument).

    • Sa — Calculate the prior covariance of the state vector (the concatenation of state vectors for the surface and radiative transfer model).

    • Sb — Accumulate the uncertainty due to unmodeled variables within respective forward model portions.

    • eof_offset — Empirical orthogonal fucntion offset. FM wrapper in the style of xa, Sa, Seps in case we want to be able to extend this across surface, RT, and instrument

    • calc_meas — Calculate the model observation at instrument wavelengths.

    • calc_Ls — Calculate the surface emission.

    • calc_rfl — Calculate the surface reflectance.

    • calc_lamb — Calculate the Lambertian surface reflectance.

    • Seps — Calculate the total uncertainty of the observation, including up to three terms: (1) the instrument noise; (2) the uncertainty due to explicit unmodeled variables, i.e. the S_epsilon matrix of Rodgers et al.; and (3) an aggregate 'model discrepancy' term, Gamma.

    • K — Derivative of observation with respect to state vector. This is the concatenation of jacobians with respect to parameters of the surface and radiative transfer model.

    • Kb — Derivative of measurement with respect to unmodeled & unretrieved unknown variables, e.g. S_b. This is the concatenation of Jacobians with respect to parameters of the surface, radiative transfer model, and instrument. Currently we only treat uncertainties in the instrument and RT model.

    • summarize — State vector summary.

    • calibration — Calculate measured wavelengths and fwhm.

    • upsample — Linear interpolation to RT wavelengths.

    • unpack — Unpack the state vector in appropriate index ordering.

    • match_statevector
